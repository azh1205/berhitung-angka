# Berhitung Angka - Matematika Sederhana 🧮

Game matematika sederhana berbasis web dan desktop yang dibuat menggunakan **HTML**, **CSS**, dan **JavaScript** melalui framework **Electron**. Game ini dirancang untuk melatih kecepatan berhitung dengan antarmuka yang simpel dan interaktif.

---

## 🚀 Cara Menjalankan Game

Ada 2 cara

### 🌐 Versi Web 
Jika kamu hanya ingin mencoba game secara langsung tanpa instalasi tambahan:
1. Klik tombol **Download ZIP** atau download file RAR dari bagian [Releases](https://github.com/azh1205/berhitung-angka/releases).
2. Ekstrak file RAR/ZIP tersebut.
3. Cari file `index.html` dan klik kanan, lalu pilih **Open with** (buka dengan) browser favoritmu (Chrome, Edge, atau Firefox).

### 💻 Versi Aplikasi Desktop (Electron)
Jika kamu ingin menjalankan atau mengembangkan game ini sebagai aplikasi desktop:
1. Pastikan kamu sudah menginstal **Node.js** di komputermu.
2. Clone repositori ini atau download filenya.
3. Buka terminal/command prompt di dalam folder project.
4. Instal dependensi dengan perintah:
   ```bash
   npm install
   ```
5. Jalankan aplikasi dengan perintah:
   ```bash
   npm start
   ```
6. (Opsional) Untuk membuat installer (setup .exe), gunakan perintah:
   ```bash
   npm run dist
   ```

---

## 🎮 Cara Bermain

1. Klik tombol **Start Game** untuk memulai.
2. Perhatikan soal matematika yang muncul di bagian atas layar.
3. Klik atau sentuh gelembung jawaban yang benar

---

## 🛠️ Apa yang digunakan?

- **HTML5**: Struktur halaman dan elemen game.
- **CSS3**: Styling, animasi gelembung, dan efek visual.
- **JavaScript (Vanilla)**: Logika game, perhitungan, dan interaksi.
- **Electron**: aplikasi desktop untuk windows.
